# הורים ביחד — אפליקציית ניהול הסידורים

אפליקציית ווב לניהול הסידורים בין הורים גרושים: משמורת, לוח שנה, ילדים, הוצאות משותפות ותקשורת.

## הפעלה מקומית

פתח/י את `index.html` בדפדפן, או:

```powershell
cd c:\Users\user\coparent-app
python -m http.server 8080
# גלש/י ל-http://localhost:8080
```

## הגדרת התחברות

ראה **[SUPABASE_SETUP.md](SUPABASE_SETUP.md)** — הגדרת Supabase להרשמה והתחברות.

## העלאה לשרת

ראה **[DEPLOY.md](DEPLOY.md)** — הדרכה מלאה ל-Netlify, Cloudflare, GitHub Pages ו-Vercel.

**הדרך המהירה:** גרור/י את התיקייה ל-https://app.netlify.com/drop

## תכונות

- **התחברות והרשמה** — חשבון משתמש אישי (Supabase Auth)

- **לוח בקרה** — סקירה: משמורת היום, אירועים קרובים, הוצאות ממתינות
- **לוח שנה** — תצוגה חודשית עם צבעי משמורת ואירועים
- **משמורת** — שבועות מתחלפים / לוח שבועי קבוע
- **ילדים** — פרופילים (גיל, בית ספר, אלרגיות)
- **אירועים** — רופא, חוגים, פעילויות
- **הוצאות** — מעקב וחלוקה בין ההורים
- **הודעות** — יומן תקשורת בין ההורים
- **הגדרות** — שמות הורים, ייצוא/ייבוא JSON

## מובייל

- עיצוב רספונסיבי מלא (RTL)
- תפריט תחתון לניווט מהיר
- תמיכה ב-safe area (iPhone)
- ניתן להוסיף למסך הבית (PWA)

## שמירת נתונים

הנתונים נשמרים ב-`localStorage` של הדפדפן. ניתן לייצא גיבוי JSON מהגדרות.

## טכנולוגיות

HTML, CSS, JavaScript — ללא צורך ב-Node.js.
